# pwnable.kr - fd

pwnable.kr의 첫 번째 문제인 fd입니다.

## 문제 분석

파일 디스크립터(File Descriptor)에 대한 이해가 필요한 문제입니다.

- 0: stdin
- 1: stdout
- 2: stderr

## 풀이

```c
int fd = atoi(argv[1]) - 0x1234;
```

0x1234는 10진수로 4660입니다. fd가 0이 되려면 argv[1]에 4660을 입력하면 됩니다.

```bash
./fd 4660
```
